# 🤖 DIO Agent Task Manager

Agente organizador de tarefas integrado ao Trello, desenvolvido com Python, Google ADK e Gemini API.

O projeto permite criar, listar, atualizar status, priorizar e remover tarefas em um board do Trello utilizando linguagem natural.

## 🚀 Tecnologias utilizadas

- Python 3
- Google ADK
- Gemini API
- Trello API
- Python-dotenv
- py-trello

## 📌 Funcionalidades

✅ Criar tarefas no Trello  
✅ Listar tarefas por status  
✅ Atualizar status das tarefas  
✅ Definir prioridade usando labels  
✅ Remover tarefas  
✅ Organização de tarefas via linguagem natural
